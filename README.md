# InfCall

InfCall is a modern caller identification mobile application built using Flutter.  
The project is designed to replicate the core user experience of a caller ID system with a clean interface and scalable architecture.

Developed by Lucky Chouhan.

---

## Overview

InfCall provides a structured foundation for a caller identification system.  
The application demonstrates UI design, state management, and modular project architecture using Flutter.

This project is intended for learning and portfolio purposes.

---

## Features

- Phone number search functionality  
- Caller name display  
- Spam and Verified tagging system  
- Recent calls interface  
- Modern dark theme UI  
- Responsive layout  
- Smooth navigation transitions  
- Scalable architecture for future backend integration  

---

## Technology Stack

- Flutter  
- Dart  
- Material 3  
- Provider (State Management)
